# 如何使用GitHub



## 前言

GitHub 是一个面向开源及私有软件项目的托管平台，因为只支持 Git 作为唯一的版本库格式进行托管，故名 GitHub。

> [!note]由于GitHub属于「国外网站」，国内直接访问速度可能会很慢，需要足够的耐心等待。
>
> 有条件可以使用技术进行「科学上网」。



## 敲开 GitHub 的大门 · 注册账号
