## Starship是什么

本站是基于 [Material for Mkdocs](https://squidfunk.github.io/mkdocs-material/) 构建的开放知识库。

**Starship**旨在为中学阶段的学生提供知识分享平台，我们欢迎愿意分享知识的同学参与丰富我们的内容。[如何贡献？](/intro/contribute/)
